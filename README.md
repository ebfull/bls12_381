# bls12_381 [![Crates.io](https://img.shields.io/crates/v/bls12_381.svg)](https://crates.io/crates/bls12_381) #

This is a pure Rust implementation of the BLS12-381 pairing-friendly elliptic curve group.

* **This implementation has not been reviewed or audited. Use at your own risk.**
* This implementation targets Rust `1.31` or later.
